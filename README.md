# Pre-ICFES Saber 11 MPB

[View Live Project](https://preicfes-plataforma.vercel.app/)

## Overview
Pre-ICFES Saber 11 MPB is an educational platform designed to empower 11th-grade students at the Institución Educativa Misael Pastrana Borrero in Cúcuta, Colombia. This project serves as the final capstone for the Technical Degree in Software Programming at SENA.

## The Problem
Many students in the public education system face a significant barrier: the lack of organized, high-quality study materials to prepare for the Saber 11 state exam. Without a centralized resource, students often feel overwhelmed or unprepared for this high-stakes assessment, which is critical for accessing higher education.

## The Solution
This platform centralizes essential academic resources into a single, user-friendly hub. By providing structured study modules and integrated practice exams, it democratizes access to quality preparation, allowing students to study at their own pace and build confidence before the official exam.

## Key Features
* Subject-Specific Modules: Organized study material covering all core areas of the Saber 11 exam.
* Real-Time Community Counter: A live tracker showing the growing number of registered students, powered by Firebase.
* Integrated Simulations: Direct access to practice exams via Google Forms for immediate self-assessment.
* Mobile-First Design: A fully responsive interface optimized for smartphones, ensuring students can study anywhere.

## Tech Stack
* HTML5: Semantic structure for accessibility.
* Tailwind CSS: Modern, utility-first styling for a clean UI.
* JavaScript (ES6+): Dynamic functionality and DOM manipulation.
* Firebase: Real-time Firestore for data synchronization.
